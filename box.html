Components

Arduino: Although we used an Elegoo-branded board, it's commonly referred to as an "Arduino" since it functions the same way and is compatible with the Arduino ecosystem..

A solderless breadboard: Used for making electronic circuits quickly without permanent connections.

Cables: Used to make quick, flexible connections between components on a solderless breadboard, enabling fast circuit assembly without soldering.

Ultrasonic sensor: measures distance by emitting sound waves and detecting their echo.

USB cable: used to power and upload code to the Arduino board.

Light Emitting Diodes: used to visualize outputs or signals in a circuit.

A potentiometer: lets you control things like brightness or speed. Turning it changes how much power flows through a circuit.

This remote lets you send signals to the Arduino wirelessly, like pressing buttons to control lights, sounds, or movement in a project.

emperature sensor for measuring environmental data in real-time projects.
